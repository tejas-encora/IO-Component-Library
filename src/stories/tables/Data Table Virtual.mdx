# Data Table Virtual

The IODataTable* components are used for displaying tabular data. Features include sorting, searching, pagination, grouping, and row selection.

The IODataTableV component relies on all data being available locally. But unlike the standard data-table it uses virtualization to only render a small portion of the rows. This makes it well suited for displaying large data sets. It supports sorting and filtering, but not pagination.
