# ioUI Component Library

## Get Started

### 1. Install

`npm install @encoradxp/mat-lib-test`

### 2. Basic Usage

```jsx
<script setup>
import IOButtonFilled from '@encoradxp/mat-lib-test
</script>

<template>
  <main>
    <IOButtonFilled label="Username"/>
  </main>
</template>
 
```
