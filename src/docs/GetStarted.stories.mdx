# ioUI Component Library

[NPM Package](https://www.npmjs.com/package/@encoradxp/mat-lib-test) | Current: 0.3.3
 
## Get Started

### 1. Install
 
`npm install @encoradxp/mat-lib-test`

**Note**: Currently work in progress: Filter Chips, DataTables. [Go to labs](?path=/docs/ui-labs--docs).

### 2. Basic Usage

```jsx
<script setup>
import IOButtonFilled from '@encoradxp/mat-lib-test
</script>

<template>
    <IOButtonFilled>Click things</IOButtonFilled>
</template>
 
```
